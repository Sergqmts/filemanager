# Файловый Менеджер на Python

## Описание
Этот проект представляет собой простой файловый менеджер, написанный на Python с использованием библиотеки Tkinter для создания графического интерфейса. Пользователь может открывать, копировать, вставлять и удалять файлы с помощью этого приложения.

## Установка
Для работы с проектом необходимо установить Python и библиотеку Tkinter. Tkinter обычно устанавливается вместе с Python, однако убедитесь, что он доступен.

Если ошибки не возникло, то библиотека доступна.

## Как запустить
1. Скачайте проект на свой компьютер.
2. Сохраните код в файл, например `main.py`.
3. Откройте терминал и перейдите в директорию, где находится файл.
4. Запустите приложение командой:
        python file_manager.py


## Использование
После запуска приложения на экране отобразится окно с четырьмя кнопками:

- **Открыть файл**: Позволяет выбрать файл для просмотра его содержимого. После выбора файла его содержимое будет отображено в диалоговом окне.

- **Скопировать файл**: Позволяет выбрать файл для копирования. Путь к выбранному файлу сохраняется в приложении.

- **Вставить файл**: Копирует сохранённый файл в текущую директорию, если файл был скопирован ранее.

- **Удалить файл**: Позволяет выбрать файл для удаления. После выбора, появляется подтверждение для удаления файла. Если вы подтвердите, файл будет удалён.

## Примечания
- Убедитесь, что у вас есть достаточно прав для открытия, копирования и удаления файлов в выбранной директории.
- Приложение не имеет расширенных функций обработки ошибок. Убедитесь, что выбранные файлы доступны.
